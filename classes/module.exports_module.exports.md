# module.exports_module.exports

## exports

### new exports(key)

此类表示 ECDSA 密钥对的私钥或公钥。

- 参数

| 名称 | 类型   | 描述                                                                                   |
| ---- | ------ | -------------------------------------------------------------------------------------- |
| key  | Object | 这必须是 jsrsasign.KEYUTIL.generateKeypair()生成的对象的"privKeyObj"或" pubKeyObj"部分 |
