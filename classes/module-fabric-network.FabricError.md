# fabric-network.FabricError

## [fabric-network](https://hyperledger.github.io/fabric-sdk-node/release-1.4/module-fabric-network.html)~ FabricError

Fabric 的基本错误类型。

#### new FabricError()

- 属性

| 名称          | 类型   | 必要性 | 描述                     |
| ------------- | ------ | ------ | ------------------------ |
| cause         | Error  | 可选   | 潜在的导致此错误的错误。 |
| transactionId | string | 可选   | 关联交易的 ID。          |
